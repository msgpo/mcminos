The desktop version of McMinos has optional keyboard shortcuts so you can play McMinos using the keyboard only.

key item/function
1 chocolate
2 key
3 bomb
4 dynamite
5 mine
6 umbrella
7 medicine
8 --
9 take a screenshot
0 switch menus on/off
