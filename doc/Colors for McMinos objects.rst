---------------------

:Authors:
  Andreas Neudecker <zapyon@gmx.net>,
  Ulrich Norbisrath <ulno@ulno.net>

* Format is RGBA: rrggbbaa
* all objects should have an outline 1 pixel wide, and black (000000ff)
